## Architecture: Complexity Checker

```
ComplexityAnalyzer/
├── CognitiveComplexityVisitor.swift  — AST visitor, deterministic scoring
├── BigOEstimator.swift               — Loop depth + stdlib cost estimation
├── PatternDetector.swift             — Anti-pattern recognition (6 patterns)
├── CallGraphBuilder.swift            — Intra-module call graph from AST
├── CallGraphAmplifier.swift          — Cross-function cost composition
├── StdlibCostTable.swift             — Built-in + user-defined cost lookup
└── ComplexityTelemetryEmitter.swift  — Corpus artifact builder
```

Data flows: Source → ComplexityAnalyzer → FunctionComplexityRecord → ComplexityReport → Corpus → PulseRefiner → ComplexityTrend/ViolationCluster

## Configuration

```yaml
complexity:
  cognitiveThreshold: 15        # flag functions above this
  reportTopN: 10
  moduleThresholds:             # per-module overrides
    Parser: 25
  emitToCorpus: true
  callGraphEnabled: true
  callGraphMaxDepth: 1
  knownCosts:                   # user-declared function costs
    - pattern: "DatabaseClient.fetch"
      cost: "O(n)"
```

CLI: `quality-gate --check complexity --threshold 20`
