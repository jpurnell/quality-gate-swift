## How To Build & Deploy

```bash
swift build -c release
sudo cp .build/release/quality-gate /usr/local/custom/bin/quality-gate
sudo codesign --force --sign - /usr/local/custom/bin/quality-gate
```

macOS 26.5 invalidates ad-hoc signatures when copying binaries — the `codesign` step is mandatory.

## How To Run the Dashboard

```bash
quality-gate dashboard              # interactive TUI (live reloads every 30s)
quality-gate dashboard --summary    # one-shot text summary
quality-gate dashboard --output-format json  # JSON output
quality-gate dashboard --project quality-gate-swift  # single project
```

## How To Run Quality Gate

```bash
quality-gate --check all --exclude test --exclude doc-lint --strict --continue-on-failure
```

## Gotchas

- **macOS codesign:** Always re-sign after `cp`. Without it, the binary crashes with "killed: 9".
- **Partial telemetry runs:** Some projects have runs with only 1 checker (e.g., release-readiness). `ProjectSummary.latestCheckerPassed` handles this by scanning runs newest-first per checker.
- **SwiftCLIKit `RawTerminal.readByte()` blocks:** The dashboard uses `poll()` on STDIN_FILENO with a 500ms timeout to enable periodic live reload.
- **disk-clean checker:** Deletes `.build/` directory. After running quality-gate, you need to rebuild.
- **swift-tools-version:** Local is 6.3, remote servers may be 6.0.x — check before deploying.
